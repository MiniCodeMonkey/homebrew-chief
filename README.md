# Homebrew Tap for Chief

This is the official [Homebrew](https://brew.sh) tap for [Chief](https://github.com/MiniCodeMonkey/chief).

## Installation

```bash
brew install minicodemonkey/chief/chief
```

Or add the tap first:

```bash
brew tap minicodemonkey/chief
brew install chief
```

## Updating

```bash
brew upgrade chief
```

## About Chief

Chief is an autonomous agent that transforms PRDs into working code using Claude Code.

- [Documentation](https://minicodemonkey.github.io/chief/)
- [GitHub Repository](https://github.com/MiniCodeMonkey/chief)
